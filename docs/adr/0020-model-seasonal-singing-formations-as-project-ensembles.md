# Model seasonal singing formations as Project Ensembles

CSK Choir Hub will model Turnekören and similar temporary singing formations as Project Ensembles rather than permanent Choirs or committee-like Groups. Project Ensemble participation belongs to the defined project or period and does not replace a User's Home Choir, Section Placement, or standing Group Memberships.

This preserves MK, KK, and DK as the three permanent Choirs while allowing seasonal formations to maintain their own participants, rehearsal activity, and voice registration.

Each spring's Turnekören is its own dated historical Project Ensemble instance, such as Turnekören 2026, rather than another active period of one timeless record. This keeps each season's roster, activity, and history self-contained.
